OWASP Node Goat Project

    • The OWASP Node Goat is an educational Node.js web application vulnerable to the OWASP Top 10 risks.

      It is intended to show how each of these vulnerabilities can manifest in a Node.js specific way, and provides the subsequent mitigation for each with source code examples.
